HR Employee Attrition Dashboard

Project Overview
This Power BI dashboard analyzes employee attrition patterns across departments, job roles, age groups, and gender.

Dashboard Metrics
* Total Employees: 1470
* Employees Left (Attrition): 237
* Attrition Rate: 16.12%

Visualizations
* Attrition by Job Role
* Attrition by Department
* Attrition by Age Group
* Department Slicer
* Gender Slicer
* Age Group Slicer

Tools Used
* Power BI
* DAX
* HR Employee Attrition Dataset

Key Insights
* Research & Development has the highest attrition.
* Laboratory Technicians show the highest attrition among job roles.
* Employees aged 26–35 have the highest attrition rate.
